Wp Auto Reload Widgets plugin allow you to refresh all of widgets in period of time. Set time in seconds and automatically refresh.

== Description ==
WP Auto Reload Widgets Plugin is allow you to refresh your widgets period of time. Not only text widgets it is support all wordpress widgets. Set the time in seconds and automatically reload / refresh your widgets. Once Installed pugin find \"Auto Reload Widget Settings\" under settings on wordpress dashboard.

== Installation ==
1. Install the plugin via the WordPress.org plugin directory, or download the plugin
2. Upload the files to your server
3. Activate the plugin through the \'Plugins\' menu in WordPress
4) After activating the plugin find the settings on \"Auto Reload Widget Settings\"
